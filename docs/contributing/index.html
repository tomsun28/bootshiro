Contributing to Sureness

Very welcome to Contribute this project, go further and better with sureness. Firstly, thanks for your interest in contributing! I hope that this will be a pleasant first experience for you, and that you will return to continue contributing.

Components of Repository:

  • sureness's kernel code--sureness-core
  • sureness integration springboot sample(configuration file scheme)--sample-bootstrap
  • sureness integration springboot sample(database scheme)-sample-tom
  • sample projects using sureness in each framework(javalin,ktor,quarkus)--samples
